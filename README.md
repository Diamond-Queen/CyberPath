# CyberPath
(planning phase)

# Mission

Help high school students develop their own knowledge on what
 cybersecurity path they wish to go on.

# User 

This app is for high school students interested in 
cybersecurity, but overwhelmed on what to learn.

# Problem
Many aspiring high school students want to enter into 
the cybersecurity profession, but have a hard time keeping track of
what they need to know as well as what steps they need to overcome. 

# Solution
CyberPath provides:
- Career assessment quiz
- Personalized cybersecurity roadmaps
- Skill tracking
- Goal-tracking 

## Features 
- Goal-Based Planning 
- Milestones
- Progress Tracking
- Accountability 

# PHASE: 1 

1. User chooses a goal
2. Cyberpath displays:
   - required skills
   - milestones
   - suggested order
3. User can mark milestones complete 
4. Progress bar updates
